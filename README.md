# Ty Bills App

## Phase 1 Goal
Build a simple bill tracking app that allows people to add bills, view bills, mark bills as paid, and see monthly totals. 

## Phase 1 Features 
- Add a bill
- View all bills
- Mark bill as paid/unpaid 
- Show monthly total 
- Show remaining unpaid balance

# Tech Stack
- Python 
- Flask
- SQLite
- HTML
- CSS

## Future Features 
- Recurring bills
- Login system
- Charts
- Payment Reminders
- Annual Projections
